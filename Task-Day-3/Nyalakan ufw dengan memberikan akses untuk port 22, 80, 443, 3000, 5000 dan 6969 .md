## Nyalakan akses port 22, 80, 443, 3000, 5000, 6969.
Dengan menggunakan command
- sudo ufw allow 22
- sudo ufw allow 80
- sudo ufw allow 443
- sudo ufw allow 3000
- sudo ufw allow 5000
- sudo ufw allow 6969
![ufw](scr/Foto-4-0.png)

Mengecek Status port yang sudah dinyalakan
- sudo ufw status
![ufw](scr/Foto-4-1.png)
